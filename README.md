# TP02_2C_Bernardi_ShooterGame

Camp Assault es un juego shooter en primera persona desarrollado en Unity, donde deberás infiltrarte en territorio enemigo y eliminar todos los campamentos hostiles para completar la misión.

Tu objetivo es explorar un amplio escenario, enfrentarte a enemigos armados y utilizar estratégicamente tus recursos para sobrevivir mientras destruyes cada campamento enemigo.

Administrá tu salud, explorá el mapa y eliminá todas las amenazas para alcanzar la victoria.

## 🕹️ Controles

| Acción           | Tecla / Botón     |
| ---------------- | ----------------- |
| Mover personaje  | `W / A / S / D`   |
| Saltar           | `Espacio`         |
| Sprint           | `Shift Izquierdo` |
| Agacharse        | `Ctrl Izquierdo`  |
| Apuntar          | `Mouse`           |
| Disparar         | `Click izquierdo` |
| Recargar         | `R`               |
| Recoger botiquín | `E`               |
| Usar botiquín    | `Q`               |
| Pausa            | `Esc`             |

## ⚔️ Mecánicas Principales

🔫 **Sistema de combate:**

* Disparos en tiempo real contra enemigos.
* Gestión de munición mediante recarga manual.
* Combate dinámico en escenarios abiertos.

❤️ **Sistema de curación:**

* Recolección de botiquines distribuidos por el mapa.
* Recuperación de vida utilizando los botiquines almacenados.

🌎 **Exploración:**

* Amplio escenario para recorrer libremente.
* Diferentes zonas para investigar y descubrir.

🏕️ **Campamentos enemigos:**

* Múltiples campamentos distribuidos por el mapa.
* Cada campamento eliminado acerca al jugador a la victoria.

🏃 **Movilidad táctica:**

* Sprint para desplazamientos rápidos.
* Agacharse para reducir la exposición al enemigo.
* Saltar obstáculos y acceder a nuevas áreas.

🏁 **Objetivo del juego:**

* Eliminar todos los campamentos enemigos del mapa.
* Sobrevivir utilizando los recursos disponibles y completar la misión.
